🌊 O QUE É UMA RELAÇÃO?

Uma relação matemática R em A é um subconjunto do produto cartesiano A×A — ou seja, um conjunto de pares ordenados (a,b).

A Armada GAR usa padrões matemáticos para coordenar suas frotas. Cada padrão é uma relação entre naves. No tabuleiro: linhas e colunas = números das naves. Célula ⚓ marcada = aquelas duas naves se comunicam.

🔁 REFLEXIVA

(a,a) ∈ R para todo a ∈ A

→ Diagonal principal toda marcada

🔄 SIMÉTRICA

Se (a,b) ∈ R então (b,a) ∈ R

→ Tabuleiro espelhado na diagonal

⛓ TRANSITIVA

Se (a,b) e (b,c) ∈ R então (a,c) ∈ R

→ Cadeia de comunicação fechada

EQUIVALÊNCIA

Reflexiva + Simétrica + Transitiva

→ As três ao mesmo tempo!

🕹 COMO JOGAR

  • ▸ O tabuleiro usa números nas linhas E colunas — ambos do conjunto A = {1,2,3,4}.
  • ▸ Cada célula = par ordenado (a,b). Ex: célula linha 2, coluna 3 = par (2,3).
  • ▸ Clique para marcar/desmarcar pares. Células com borda tracejada são fixas.
  • ▸ Clique DECODIFICAR para checar. Células ficam verdes ✓ ou vermelhas ✗.
  • ▸ Use 💡 DICA para consultar o caderninho do Willy com a explicação matemática.
  • ▸ +20 pts sem erros · +10 pts com erros · -5 pts por tentativa extra.
